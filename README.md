## 🎯 ¿Para qué sirve este repositorio?

Este repositorio fue creado con el propósito de **centralizar todas las prácticas, ejercicios y recursos** utilizados durante las clases del bootcamp. Está pensado para facilitar el aprendizaje en equipo entre los **compañeros docentes** participantes del programa MINEDUCYT.

Aquí encontrarás:

- ✅ Código fuente de todos los ejercicios realizados en clase
- ✅ Materiales de referencia por tecnología
- ✅ Proyectos prácticos y mini-proyectos
- ✅ Recursos adicionales de apoyo

## 👩‍🏫 ¿Cómo usar este repositorio?

### 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/bootcamp-mineducyt.git
```

### 2️⃣ Navegar a la práctica deseada

```bash
cd bootcamp-mineducyt/01-HTML/practica-01-estructura-basica
```

### 3️⃣ Abrir con tu editor favorito

```bash
code .   # VS Code
```

## 🤝 Contribuir al Repositorio

¡Este repositorio es colaborativo! Si eres docente del bootcamp y deseas agregar o mejorar contenido, sigue estos pasos:

1. 🍴 Haz un **fork** del repositorio
2. 🌿 Crea una rama nueva: `git checkout -b mejora/nombre-practica`
3. 💾 Realiza tus cambios y haz commit: `git commit -m "Agrega práctica de flexbox"`
4. 📤 Sube tus cambios: `git push origin mejora/nombre-practica`
5. 🔁 Abre un **Pull Request**

## 📋 Convención de Nombres

Para mantener el orden, se recomienda seguir esta convención al subir prácticas:

```
practica-[número]-[descripción-corta]/
  ├── index.html
  ├── style.css
  ├── script.js
  └── README.md  ← (opcional, pero muy valorado 💙)
```

- 💬 Comenta tu código para facilitar el aprendizaje de los demás
